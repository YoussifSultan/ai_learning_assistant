1. What is a limit?

In calculus, a limit describes the value that a function approaches as the input (usually xx) gets closer and closer to a certain point.

Think of it like this:
👉 “As xx gets closer to some value aa, what does f(x)f(x) get closer to?”

We write this as:

lim⁡x→af(x)=L\lim_{x \to a} f(x) = L

which means: as xx approaches aa, f(x)f(x) approaches LL.

2. Simple example

lim⁡x→2(x2+3)=?\lim_{x \to 2} (x^2 + 3) = ?

Plugging in:

x2+3when x=2→22+3=7x^2 + 3 \quad \text{when } x=2 \quad \to 2^2 + 3 = 7

So:

lim⁡x→2(x2+3)=7\lim_{x \to 2} (x^2 + 3) = 7

3. When limits are useful

Sometimes, you can’t just “plug in” the value. For example:

lim⁡x→0sin⁡xx\lim_{x \to 0} \frac{\sin x}{x}

Direct substitution gives 00\frac{0}{0}, which is undefined.
But the limit exists, and it equals 1.
This is why limits are powerful — they let us define values even when substitution fails.

4. Types of limits

  • Finite limits → Approach a specific number.
    Example: lim⁡x→3(2x+1)=7\lim_{x \to 3} (2x+1) = 7.

  • Infinite limits → Function grows without bound.
    Example: lim⁡x→0+1x=+∞\lim_{x \to 0^+} \frac{1}{x} = +\infty.

  • One-sided limits → Approach from left or right.
    Example:
    lim⁡x→0+1x=+∞\lim_{x \to 0^+} \frac{1}{x} = +\infty
    lim⁡x→0−1x=−∞\lim_{x \to 0^-} \frac{1}{x} = -\infty.

5. Why limits matter

  • They are the foundation of derivatives (rate of change).

  • They define continuity (a function is continuous if the limit exists and matches the function value).

  • They help deal with infinity and indeterminate forms (0/0,∞/∞0/0, \infty/\infty, etc.).

✅ Summary:
A limit tells you what value a function gets closer to as the input approaches some number. If it settles around a value (finite or infinite), the limit exists. If it “jumps” or oscillates, the limit may not exist.
